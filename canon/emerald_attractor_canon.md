# Emerald Attractor Canon

## Canonical form

```text
Unity -> Law -> Multiplicity -> Paradox -> Alignment -> Higher Unity
```

## Operator form

```text
P -> A -> Π -> Π -> A
```

## Bindings

- **P** = Prime / Seed / Unity
- **A₁** = Axiom / Law / Constraint
- **Π₁** = Multiplicity / Branching / Domains
- **Π₂** = Paradox / Opposition / Reflective Split
- **A₂** = Alignment / Reintegration

## Core rule

> Do not average contradiction away. When paradox appears, first test whether it indicates a missing dimension, regime, timescale, representation, or constraint. If a lawful missing axis exists, perform a dimensional lift. If no lawful lift exists, preserve the contradiction explicitly as darkness or fracture.

## State machine

1. Seed
2. Law
3. Branch
4. Paradox
5. Lift or conflict
6. Alignment or darkness
7. Higher unity or fracture record

## Resolution modes

### Alignment

A lawful missing axis exists. The contradiction becomes structure in a lifted representation.

### Darkness

The conflict is not honestly resolved yet. Preserve uncertainty and name the minimum next observation.

### Fracture

The literal constraints cannot jointly hold. Record the boundary and collapse trigger.
