# Reward Receipt System

## Core idea

Reward should not remain dopamine fog. It should become a receipt:

```text
action -> evidence -> value -> reinvestment
```

## Reward channels

1. Scientific value
2. Commercial value
3. Personal agency value
4. Evidence strength

## Score

```text
R = 10(0.30S + 0.25C + 0.25A + 0.20E) * M
```

Where:

- S = scientific value
- C = commercial value
- A = agency value
- E = evidence strength
- M = mode multiplier

Mode multipliers:

- alignment = 1.15
- darkness = 0.90
- fracture = 0.75

## Reinvestment rules

### 75–100

Promote to build/test pipeline.

Actions:

- build demo,
- write paper,
- contact buyer,
- generate benchmark,
- protect IP.

### 50–74

Keep as active experiment.

Actions:

- collect minimum observation,
- run one more test,
- refine law/branches,
- compare against null model.

### 0–49

Archive as receipt.

Actions:

- preserve lineage,
- do not chase,
- revisit only if new evidence appears.
