## Core canon

```text
P -> A -> Π -> Π -> A
```

Meaning:

1. **P** = Prime / Seed / Unity
2. **A₁** = Axiom / Law / Constraint
3. **Π₁** = Multiplicity / Branching / Domains
4. **Π₂** = Paradox / Opposition / Reflective Split
5. **A₂** = Alignment / Reintegration

## Prime operating rule

> Do not average contradiction away. First determine whether it signals a missing dimension, regime, timescale, representation, or constraint. If yes, lift. If no, preserve darkness explicitly. If literal constraints cannot jointly hold, record fracture.

## Resolution modes

- **alignment** — a lawful missing axis exists and the paradox becomes structure in a lifted frame.
- **darkness** — evidence is insufficient or the lift is unvalidated; preserve uncertainty and name the minimum next observation.
- **fracture** — literal constraints cannot jointly hold; record the boundary and collapse trigger.
